The schedule page (`about.html`) displays Patrick's Spring 2024 university class schedule as a simple timetable. It is a standalone page separate from the main portfolio.

<Note>
  The original HTML file contains a typo in the `<title>` tag: **"Sping 2024 Class Schedule"** instead of "Spring 2024 Class Schedule". This typo is present in the source and appears in the browser tab.
</Note>

## Navigation

A fixed anchor in the top-right corner links back to the main portfolio page. It uses the same `.schedule` class as the home page's nav link, so both pages share a consistent button appearance:

```html about.html
<a class="schedule" href="index.html">
    Home
</a>
```

The `.schedule` class is defined in `index.css` and positions the element fixed at `top: 15px; right: 30px` with an indigo (`#4C3AFF`) background.

## Stylesheets

The schedule page loads two CSS files. `about.css` is listed first, followed by `index.css`:

```html about.html
<link rel="stylesheet" href="css/about.css">
<link rel="stylesheet" href="css/index.css">
```

Because `index.css` is loaded second, its rules take precedence when selectors conflict. In practice the two files target distinct elements — `about.css` styles the schedule container and table, while `index.css` styles the shared `.schedule` nav button — so there is no meaningful conflict.

## Dark theme

The schedule page inherits the same dark theme used in the hero section of the home page:

| Property | Value |
|----------|-------|
| Background | `#212529` (dark charcoal) |
| Text | `#ffffff` (white) |
| Accent / nav button | `#4c3aff` (indigo) |

<Tip>
  The shared color variables in `index.css` mean that updating `#212529` or `#4c3aff` in one place will apply consistently across both `index.html` and `about.html`.
</Tip>
