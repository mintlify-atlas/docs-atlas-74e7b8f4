### Animated heading

Each word in "Hello, I'm Patrick" has its own color defined in CSS custom properties, and a `::before` pseudo-element produces the color-fill reveal animation on hover.

```html index.html
<h1
  class="heading"
  data-text="Hello,"
  style="--clr: #ffdd1c"
>
  Hello,
</h1>
<h1
  class="heading"
  data-text="I'm"
  style="--clr: #40BB94"
>
  I'm
</h1>
<h1
  class="heading"
  data-text="Patrick"
  style="--clr: #4567DB"
>
  Patrick
</h1>
```

The `data-text` attribute is read by the `::before` pseudo-element so it can overlay a colored copy of each word and clip-reveal it from left to right on hover.

```css index.css
.heading::before {
  content: attr(data-text);
  color: var(--clr);
  position: absolute;
  top: 0;
  left: 0;
  width: 0;
  overflow: hidden;
  transition: 1s;
  border-right: 8px solid var(--clr);
  white-space: nowrap;
}

.heading:hover::before {
  width: 100%;
}
```

<Tip>
  The `border-right` on the `::before` element acts as a visible cursor that moves across the text as the `width` animates from `0` to `100%`, reinforcing the left-to-right reveal effect.
</Tip>

### Wave divider

An SVG wave shape separates the hero from the work experience section. It transitions between `#212529` (hero background) and `#FFFFFF` (work section background) using two `<path>` fills, creating a seamless visual handoff.

The work experience section (`class="busSect"`) uses a two-column grid (`grid-template-columns: 1fr 1fr`). Each card has rounded corners (`border-radius: 30px`), a fixed height of `550px`, and a soft drop shadow.

```css index.css
.card {
  box-shadow: 0px 20px 50px 0px rgba(0, 0, 0, 0.2);
  border-radius: 30px;
  height: 550px;
}
```
